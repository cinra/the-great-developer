## 解説

### MVxとは

ソフトウェア開発のデザインパターンの事で、まとめてMVxなどと呼びます。

- MVC（Model-View-Controller）
- MVVM（Model-View-ViewModel）
- MVP（Model–View–Presenter）

など複数存在します。

### MVxフレームワークとは

目的のデザインパターンを組むためのベースとなるパッケージの事です。  
フロントエンドでは、以下のようなものが存在します。

- Vue.js
- React.js
- Riot.js
- Angular.js
