# Frontend > Basic > CSS

## 解説

CSS（カスケーディングスタイルシート）とは、HTMLに装飾を加えるための言語です。  
ウェブページを表示した際に「文字サイズ」や「色」、また「要素間のスペース」など、見た目に関わる部分を整える役割を担っています。
